## How to Run This Bot
### Prerequisites
* Python 3.7
* pandas
* numpy


### Setup
* Clone this repo from github
* Install dependencies: `pip install pandas`
* Install dependencies: `pip install numpy`
* Install dependencies: `pip install matplotlib`
* Install dependencies: `pip install PyQt5`


### Run
To run the app use:
    run python PathsViewerGUI.py
